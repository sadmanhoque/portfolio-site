Overview

I was somewhat new to terraform at the time and wanted to build a sort of reference point for creating a basic single container react site and have it deployed on AWS and be usable.

While doing that I also took the opportunity try out Kubernetes which I was completely new to back then, only familiar with ECS so far, and have an automated deployment system setup using github actions.

The Challenge

The biggest difficulty was getting Fargate to work with EKS, turns out at the time there was just a bug on AWS where the proper setup, as documented officially, simply would not work. The benefit of Fargate is that it is completely self-managed in terms of scaling.
