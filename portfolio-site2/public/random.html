Google Maps Randomizer

2023
Practice
Personal Project
ReactJS HTML Goland Docker

Overview

Ever been annoyed at the idea of wanting to get takeout and thinking like, we know we want pizza or burger or something but can't seem to decide what to get? Basically this just spits out a random result for whatever search term.

Initially this was about food, but after making it I realized it actually works for anything, restaurants, barbers, things to do, etc.

The Challenge

Mostly just to play with ReactJS and HTML, as well as an excuse to try out a functional docket container and bit of a Golang project. The Go version I never completed but React one works just fine.

Process

01

Research

I had to do a bit of Googling on figuring out which Google Maps API I needed to use and how to use it.

02

Design

This was made for me, and I hate websites asking for my location and I'm a bit of a map nerd. So we need to provide map coordinates, a range and then search term to get random results.

03

Build

The version that works is made using ReactJS, I did start a Golang version but never finished that bit.

04

Ship & Iterate

I doubt anybody would want to pay for this, but maybe someday it can be a part of a profitable service, we need more whimsy in webapps.
