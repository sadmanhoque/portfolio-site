Overview

Mostly as a way for me to try out new tools through using it for something I find interesting. In this case that is the use case of using LLM to quickly find specific information from a specific source rather than chatgpt and what not grabbing things from who knows where, and ideally my tool would also provide a reference to which part of the text it got the info from.

The Challenge

I have next to no experience using LLM packages so there's that, and I also want the final tool to have a presentable albeit basic UIUX with some sort of html page.

Process

Reading through the official docs and introduction guides available on huggingface website. Extending the project to general AI use, experimenting with ollama as well.

Deciding on a narrow scope for what I want to build, in this case providing an input text, a wiki article for example, and be able to ask Qs about it.
